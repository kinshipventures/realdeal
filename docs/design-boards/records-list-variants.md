# Real Deal - Records List Variants

## Goal

Keep the table powerful, but make the page feel calmer and more designed.

## Variant A - Calm Header, Strong Table

### Idea

Make the page opener quieter and let the table become the hero quickly.

### Moves

- lighter top frame
- tighter filter strip
- utility actions move into a calmer secondary cluster
- table gets a stronger container

### Pros

- best for heavy daily use
- simplest improvement path

### Cons

- less visually distinct at the top

## Variant B - Review Workspace

### Idea

Turn the page into a review workspace with a clearer top frame and a more deliberate table shell.

### Moves

- stronger page opener with title, pulse, and view mode
- filter and utility controls split into two rows
- table sits inside a framed workspace surface
- bulk action mode gets a more deliberate state

### Pros

- more product feel
- stronger visual hierarchy

### Cons

- slightly more complex than A

## Recommendation

Pick Variant B.

It keeps the power while making the page feel less like a raw admin table.
