# Real Deal - Campaign Detail Variants

## Goal

Make campaign detail feel like a calm operating surface.

## Variant A - Unified Top Frame

### Idea

Merge title, description, stats, and controls into one cleaner page frame.

### Moves

- one top campaign band
- stats live inside the same frame
- action controls break into primary and secondary groups
- board or table sits directly below as the hero workspace

### Pros

- strongest hierarchy
- reduces the stacked-toolbar feeling

### Cons

- requires the most cleanup

## Variant B - Layered Controls

### Idea

Keep separate bars, but make each one lighter and better grouped.

### Moves

- keep the current sequence
- tighten each bar
- reduce visual weight
- make the board area stronger

### Pros

- lower risk

### Cons

- still closer to the current stacked feel

## Recommendation

Pick Variant A.

This page needs a stronger single frame at the top more than it needs more polish on separate bars.
