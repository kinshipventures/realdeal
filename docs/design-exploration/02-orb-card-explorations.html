Orb Map Layout
Two Map Compositions
Beyond orb styling — how the whole canvas feels. Organic vs geometric, connection line treatment, depth layering.

A. Organic Layout + Gradient Connections

Orbs cluster naturally — not in a perfect circle. Connection lines fade from hub color to orb color. Sizes driven by contact count. Feels like a brain map, not a diagram.

B. Depth Layers + Glow Halos

Priority pods are larger and closer (sharper focus). Lower priority recedes (smaller, slightly faded). Each orb casts a colored glow. Feels like looking into a constellation.
Dark canvas makes colors pop. Glow halos create atmosphere. Priority orbs come forward while minor ones recede. Most "Spotify Wrapped" of the options.
