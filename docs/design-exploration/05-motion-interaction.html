Health Ring
How the Ring Fills
When the orb first appears or when equity score changes. The ring needs to communicate "this is a data value" through its motion.

A. Smooth Sweep

Ring sweeps from 0 to its value with a smooth deceleration. Clinical, precise. Like a gauge filling.
MAPS
Clean, professional. Easiest to read the final value. Best for initial load.

B. Bounce Overshoot

Ring fills past its target then settles back — like a needle bouncing on a dial. Adds personality to a data display.
TLT
More energetic. The bounce makes each ring feel like a small celebration (or warning). Adds character.
Glow Halo
How the Glow Behaves
The colored glow around each orb. Should it be static, alive, or reactive? Interact with each to feel the difference.

A. Steady Glow

Always on, never changes. Quiet confidence. The glow is environmental — it just exists.
MAPS
Most professional. Glow is a design element, not an animation. Never distracts.

B. Slow Breathe

Glow gently pulses — orb subtly scales up/down. The network feels alive, like a resting heartbeat. Staggered per orb.
MAPS
TLT
Beautiful but can feel busy with 6+ orbs all breathing. Best if only 1-2 orbs breathe (e.g. the ones needing attention).
