```typescript
const KNOWN_FIELDS: Record<string, string> = {
  'name': 'Name', 'email': 'Email', 'phone': 'Phone',
  'company': 'Company', 'role': 'Role', 'location': 'Location',
```

3. `importContacts(rows: Record<string, string>[], podId: string, onProgress?: (progress: ImportProgress) => void): Promise<ImportResult>` — Core import function:
```typescript
type ImportProgress = { current: number, total: number, imported: number, skipped: number }
type ImportResult = { imported: number, skipped: number, errors: string[] }
```
Steps:
- Fetch all existing contacts via `getContacts()` for dedup
- Build dual index: email (lowercase) and name (lowercase + trim) — same as importServiceProviders.ts lines 137-146
- For each row: check emailIndex then nameIndex. If match, skip (increment skipped count). If no match, call `createContact()` with `list_ids: [podId]`
- Map CSV fields to Contact fields using the KNOWN_FIELDS mapping
- Pace at 250ms between creates (Airtable rate limit)
- Call onProgress callback after each row
- Catch per-row errors, push to errors array, continue processing
- Do NOT update existing contacts — skip only (per D-07: "Skip duplicates, show count of skipped vs imported")
