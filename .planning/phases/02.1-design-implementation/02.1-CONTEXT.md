<decisions>
## Implementation Decisions

### Typography
- Swap Playfair Display to Fraunces 700/800 everywhere (variable font with optical size axis)
- Google Fonts link in index.html: `Fraunces:opsz,wght@9..144,400;9..144,700;9..144,800;9..144,900`
- Display: 32px/800, Heading: 18px/700, Subheading: 15px/700, Pod name: 13px/700 (all serif)
- Body: 13px/400-500, Caption: 11px/500, Score: 22-28px/700, Orb label: 8-13px/600 (all sans)

### Orb Surface
- Two-tone color shift: diagonal gradient (135deg) from base color to hue-shifted neighbor
- NOT lighter/darker of same hue — actual color shift (e.g. red → orange-red, purple → indigo)
- Pod colors from DESIGN.md: Maps=#E53935→#FF6B4A, LPs=#FF6B8A→#FF8FA5, Companies=#7E57C2→#5C6BC0, Talent=#25B439→#00BFA5, Service Providers=#F5A623→#FFD54F, Hub=#1C1C1E→#2C2C30

### Orb Effects
- Glow halo via box-shadow: pod color at 0.25 opacity at rest, 0.40 on hover
- Health ring: 2px stroke SVG arc, 4px gap from orb edge, fills on entrance with 1200ms sweep
- Ring track: rgba(0,0,0,0.06)

### Orb Sizes
- Hub: 136px, no hover scale
- Pod/List: 96px, hover scale 1.05
- Category: 60px, hover scale 1.08

### Orb Hover + Press
- Hover: scale up + translateY(-3px) + glow intensifies. 0.2s cubic-bezier(0.34, 1.2, 0.64, 1)
- Press: scale down to 0.94. 0.08s ease-in. iOS squish.
- Cursor: pointer for clickable, grab/grabbing for draggable

### Map Layout
- Concentric dashed orbit rings from hub center
- Priority pods orbit closer, lower priority further out
- Connection lines: hub-to-pod + cross-connections. Style: 1.5px rgba(0,0,0,0.06)
- Category clusters: small orbs cluster near parent pod, not scattered

### Entrance Animation
- All orbs start stacked at hub center, fly out to orbital positions with staggered delay
- Hub appears first, then pods radiate outward
- 0.7s cubic-bezier(0.22, 1, 0.36, 1), stagger: 0.1s per orb

### Dashboard Cards
- Mini orb avatar (30px) at left of each pod card — same two-tone gradient as map orbs
- Card hover: scale 1.02 + box-shadow shifts to pod color. 0.15s ease

### Empty States
- Colored orb illustration (pod color or brand green) with line icon inside
- Ghost hint cards (dashed borders) showing shape of future data
- One primary green CTA button per state
- Warm copy: "Your network starts here", "No pulse yet", "{Pod name} is empty", "No history yet", "No one by that name", "All caught up"

### Loading States
- Skeleton shimmer: ghost shapes of final UI with shimmer animation
- Orb map: single green orb pulses where hub will appear, then orbs radiate out

### Dark Mode
- Follow prefers-color-scheme, no manual toggle
- Surfaces: #111 bg, #1a1a1a surface
- Green band darkens to #1B8A2A
- Orb glows intensify (0.25 → 0.40 opacity)
- Hub gets rgba(255,255,255,0.06) border ring
- Brand green brightens to #2ECB45
- Text: white at 87%/45%/25% opacity hierarchy

### Responsive
- Desktop (960px+): full dashboard, floating pill nav, side panel
- Tablet (768px): 2-column pod grid, pill nav, side panel
- Phone (<768px): horizontal scroll pods, bottom tab bar (48px, heart=Pulse, crosshair=Map), full-screen contact sheet from bottom
- Mobile map: pinch-to-zoom, 44px touch targets, tighter orbits, tap-to-fan clusters

### Icons
- Lucide library, rounded line style, 1.5px stroke, rounded caps/joins
- Sizes: 12px inline, 16px timeline/pills, 20px nav/buttons, 28px empty states
- Active nav: solid fill in brand green. Inactive: stroke outline

### Interaction Type Colors
- Call=#2E7D32, Email=#1565C0, Text=#7B1FA2, Meeting=#E65100, Note=rgba(0,0,0,0.50), Intro=#C2185B

### Data Visualization
- Sparklines: 30-day trend in pod cards, pod-colored area fill at 12% opacity
- Health bars: horizontal bars with two-tone gradient, sorted by health
- Equity trend: 6-month line chart with green area fill
- Segmented ring: per-contact equity ring by interaction type
- Frequency bars: 8-week bar chart per contact

### Copy/Voice Updates
- Score labels: Thriving (85+), Steady (70+), Cooling (50+), Fading (<50)
- Dormancy: "Slipping away" (180+d), "Going quiet" (120+d), "Cooling off" (<120d)
- Search: "Find someone...", Notes: "What happened?", Create: "Add to network"
- Delete: "Remove this person?", Dormant action: "Let go"
- Empty: "No one here yet", Overdue: "{N}d ago", Never: "Never reached"
- Section headers: "{N} gone quiet", Success: "Logged with {name}."
- Field renames: Specialization→"Focus", Past clients→"Known for", Recommended by→"Intro'd by"
- Section rename: Relationship context→"Context"

### Accessibility
- Focus ring: 2px solid brand green, 2px offset (keyboard only)
- ARIA: role="navigation" on nav, aria-label on icon buttons, aria-current="page" on active tab, aria-live="polite" on equity score, role="dialog" on contact panel
- Touch targets: minimum 44x44px on all interactive elements
- prefers-reduced-motion: disable entrance animations and breathing effects

### Claude's Discretion

- Whether to add lucide-react as dependency or use inline SVG
- How to implement sparklines (inline SVG vs canvas vs library)
- Skeleton shimmer CSS technique
- React Flow integration approach for orbit rings and entrance animations

</decisions>

## Specific Ideas
