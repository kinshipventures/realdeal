## What Was Built

**Task 1 — Responsive nav (App.tsx):**
- `useIsMobile` hook using `window.matchMedia('(max-width: 767px)')` with change listener
- Mobile: fixed 48px bottom tab bar with Heart SVG (Pulse) and Crosshair SVG (Map) icons
- Desktop: existing floating pill nav preserved
- Both navs: `role="navigation"`, `aria-label="Main navigation"`, `aria-current="page"` on active tab
- `paddingBottom: 'env(safe-area-inset-bottom)'` for iPhone notch
- Content area gets `paddingBottom: 56` when mobile tab bar is showing

**Task 2 — Mobile contact panel (ContactPanel.tsx + globals.css):**
- `useIsMobile` hook duplicated in-component
- Mobile: `position: fixed; inset: 0` full-screen panel with `panel-slide-up` animation
- Desktop: `position: fixed; top: 0; right: 0; width: 360px` side drawer preserved
- Mobile header: back arrow (←) button instead of X close
- Desktop header: X close button preserved
- `role="dialog"` and `aria-labelledby="panel-title"` on panel wrapper
- `id="panel-title"` on category name heading
- `@keyframes panel-slide-up` + `.panel-enter-mobile` class added to globals.css
