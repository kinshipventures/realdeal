**OrbMap.tsx** - Add highlight support per D-13:
- Add prop: `highlightedPodIds?: string[]`
- Update export: `export function OrbMap({ highlightedPodIds }: { highlightedPodIds?: string[] })`
- Add state: `const [activeHighlights, setActiveHighlights] = useState<Set<string>>(new Set())`
- Add effect that watches `highlightedPodIds`:
  ```
  useEffect(() => {
    if (!highlightedPodIds?.length) return
    setActiveHighlights(new Set(highlightedPodIds))
    const timer = setTimeout(() => setActiveHighlights(new Set()), 2500)
    return () => clearTimeout(timer)
  }, [highlightedPodIds])
  ```
- After setNodes, run a setNodes updater that adds `highlighted` flag:
  ```
  useEffect(() => {
    if (activeHighlights.size === 0) return
    setNodes(prev => prev.map(n => ({
      ...n,
      data: { ...n.data, highlighted: activeHighlights.has(n.id) }
    })))
  }, [activeHighlights, setNodes])
  ```
  And when highlights clear, remove the flag the same way.

- If `mapView === 'pod'` and highlight comes in, call `drillBackToHub()` first so the pod is visible, then highlight after a short delay (500ms).
