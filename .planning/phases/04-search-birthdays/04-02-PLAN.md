```typescript
export interface BirthdayItem {
  contact: Contact

  date: string        // formatted display date, e.g. "Mar 28"
  daysUntil: number   // 0 = today, 1 = tomorrow, etc.
  isToday: boolean
}
```

**getUpcomingBirthdays function:**
```typescript
export function getUpcomingBirthdays(
  contacts: Contact[],
  pods: Pod[],
  windowDays = 14
): BirthdayItem[]
```

Logic:
1. Get today as a Date object (midnight local time)
2. Build a `Map<string, Pod>` from pods array for O(1) lookup
3. Filter contacts where `contact.birthday` is not null
4. For each contact with a birthday:
   - Parse birthday string (YYYY-MM-DD format) to extract month and day
   - Create a Date for this year with that month/day
   - If that date has already passed this year, try next year's date
   - Calculate `daysUntil` as the difference in days between that date and today (use `Math.round((target - today) / 86400000)`)
   - If `daysUntil >= 0 && daysUntil <= windowDays`, include it
   - `isToday` = `daysUntil === 0`
   - Format display date using `Intl.DateTimeFormat('en-US', { month: 'short', day: 'numeric' })` on the birthday date
   - Find pod via `contact.list_ids[0]` lookup in pods map
5. Sort results by `daysUntil` ascending (soonest first). Same-day ties sorted alphabetically by name (per Claude's discretion)
6. Return array of `BirthdayItem`

**formatDaysUntil helper (not exported, used internally and by Dashboard):**
Actually, export this for use in Dashboard row rendering:

- 0 -> "Today"

- 7 -> "1w"
- 8-13 -> "Nd"
- 14 -> "2w"

Simple implementation: if `days === 0` return "Today", else return `${days}d`.

```typescript
function BirthdayRow({ item, onClick }: { item: BirthdayItem; onClick: () => void }) {
  const podColor = item.pod?.color ?? '#718096'

  return (
    <button
      type="button"
      onClick={onClick}
      className="interactive-row"
      style={{
        display: 'flex', alignItems: 'center', gap: 12,
        width: '100%', padding: '12px 24px',
        background: item.isToday ? 'hsla(30, 80%, 55%, 0.06)' : 'none',
        border: 'none',
        borderBottom: '1px solid var(--divider)',
        cursor: 'pointer', textAlign: 'left',
      }}
    >
      {/* Pod-colored dot */}
      <div style={{
        width: 8, height: 8, borderRadius: '50%', flexShrink: 0,
        background: podColor,
      }} />

      {/* Name */}
      <div style={{
        fontSize: 13, fontWeight: 500, color: 'var(--color-text-primary)',
        flex: 1, minWidth: 0,
        overflow: 'hidden', textOverflow: 'ellipsis', whiteSpace: 'nowrap',
      }}>
        {item.contact.name}
      </div>

      {/* Date (e.g. "Mar 28") */}
      <span style={{
        fontSize: 11, color: 'var(--color-text-secondary)', flexShrink: 0,
      }}>
        {item.date}
      </span>

      {/* Days until */}
      <span style={{
        fontSize: 11, fontWeight: 500, flexShrink: 0, minWidth: 32, textAlign: 'right',
        color: item.isToday ? 'hsla(30, 80%, 55%, 0.90)' : 'var(--color-text-tertiary)',
      }}>
        {formatDaysUntil(item.daysUntil)}
      </span>

      {/* Pod name */}
      {item.pod && (
        <span style={{
          fontSize: 10, color: 'var(--color-text-tertiary)', flexShrink: 0,
          textTransform: 'uppercase', letterSpacing: '0.04em',
        }}>
          {item.pod.name}
        </span>
      )}
    </button>
  )
}
```
